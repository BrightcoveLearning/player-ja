Brightcove Player でライブ SSAI のプレーヤー拡張機能を使用する

{{ page.description }}

はじめに

ライブSSAI でのブライトコーブのプレーヤー機能強化では、広告数、広告カウントダウンタイマー、クリックスルー広告、コンパニオン広告など、ライブストリームの広告ブレイクを向上させることができます。これらのライブストリームは、Brightcove Player を使用して再生できます。

この機能を初めて使用する場合は、以下を参照してください。

  • 概要:ライブ SSAI のプレーヤーの機能強化
  • ライブ SSAI のプレーヤー機能拡張の実装

ステップ

Brightcove Player を使用してプレーヤーの機能強化を含むライブストリームを再生するには、次の手順に従います。

  1. Video Cloud Studio のプレーヤーモジュールから既存のプレーヤーを作成または選択します。

  2. Brightcove Playerは、SSAIプラグインを使用して設定する必要があります。フォローするSSAIプラグインを追加しますで説明されている手順プログラムによるSSAIの実装 SSAIプラグインドキュメントのセクション。

    Live SSAI を使用したプレーヤーの強化には、Brightcove Playerには SSAI プラグイン ( videojs-ssai ) が必要です。現在、SSAI プラグインは広告設定 ID を必要とするため、Video Cloud Studio を使用してプレーヤーに追加することはできません。
  3. のワークフローに従いますライブSSAIのプレーヤー拡張機能の実装再生トークンの作成までのドキュメント。
  4. 実装を選択します。
    • 標準埋め込みコード
    • 高度な埋め込みコード
    • ライブSSAIソース
  5. 「Live SSAI 用のプレーヤーの機能強化の実装」ドキュメントのワークフローに進みます。

標準埋め込みコード

Video Cloud Studio の Media モジュールからビデオを公開する場合、標準の埋め込みコードを使用してライブストリームを再生できます。次の手順に従ってください。

  1. Mediaモジュールから、ライブストリーミング用のリモートビデオを選択し、SSAI 対応プレーヤーで公開します。
  2. 標準の埋め込みコードをコピーし、Web ページに貼り付けます。

  3. <iframe>タグには、src属性が表示されます。この最後に、を追加しますadConfigIdあなたの値を持つURLパラメータ再生トークン。

    <iframe src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/
index.html?videoId=6089818309001&adConfigId=live.aaB61234aabbccdd567890"
  allowfullscreen=""
  allow="encrypted-media"
  width="960" height="540">
</iframe>
  4. プレーヤーはライブストリームを再生する準備が整いました。

高度な埋め込みコード

Video Cloud Studio の Media モジュールからビデオを公開する場合、高度な埋め込みコードを使用してライブストリームを再生できます。以下の手順に従います。

  1. Media モジュールから、ライブストリーミング用のリモートビデオを選択し、SSAI 対応のプレーヤーで公開します。
  2. 高度な埋め込みコードをコピーして Web ページに貼り付けます。

  3. <video-js>タグに、data-ad-config-id再生トークンの値を持つ属性を追加します。

    <video-js
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id="6089818309001"
  data-ad-config-id="live.aaB61234aabbccdd567890"
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
  4. プレーヤーはライブストリームを再生する準備ができました。

ライブSSAIソース

VOD SSAI ソースとは異なり、Live SSAI ソースには初期 VMAP は含まれません。これは、ライブSSAIストリームでは、広告ブレイクがいつ含まれるかを事前に認識しないためです。代わりに、広告ブレイクは、ライブジョブに追加したキューポイントからトリガーされます。

ブライトコーブのライブ SSAI ストリームは、特定の命名規則を持つ標準 HLS マニフェスト URL から始まります。

https://.../playlist_ssaiM.m3u8

SSAI プラグインは、_ssaiM.m3u8サフィックスと通常の HLS MIME タイプを探します

Video Cloud Studio の Media モジュールからビデオを公開する場合、ライブの SSAI ストリームを次のように再生できます。

  1. Media モジュールから、ライブストリーミング用のリモートビデオを選択し、SSAI 対応のプレーヤーで公開します。
  2. 高度な埋め込みコードをコピーし、Web ページに貼り付けます。

  3. ページコードに、次のコードを追加します。

    <video-js id="myPlayerID"
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id=""
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.src({
      type: 'application/x-mpegURL',
      src: 'https://path/to/playlist_ssaiM.m3u8'
    })
  });
</script>
  4. プレーヤーはライブストリームを再生する準備ができました。
