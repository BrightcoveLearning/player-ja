プレイヤー開発者の基本:カスタムプラグイン-Studioを使用する

このトピックでは、Studio を使用して Brightcove Player にカスタムプラグインを割り当てる方法について説明します。
 

ステップ

  1. Studio で、 PLARersモジュールを選択します。
  2. カスタムプラグインを追加するプレーヤーをクリックします。
  3. プレーヤーのプロパティで、左側で [ プラグイン] オプションを選択します。
  4. [ プラグインの追加]ドロップダウンから、[ カスタムプラグイン] を選択します。
  5. 以下を追加します。
    • プラグイン名
    • JavaScriptのURL
    • CSS URL-必要な場合
    • オプション (JSON)-必要な場合
  6. プレイヤーを保存します。
  7. プレーヤーを公開します。
