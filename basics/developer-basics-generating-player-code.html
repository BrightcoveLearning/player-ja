プレイヤー開発者の基本:プレイヤーコードを生成しています

このトピックでは、後でプレーヤーの開発に使用できるプレーヤーコードを生成する方法を学びます。
  1. Studio で、プレーヤーモジュールを開きます。
  2. [ 新規プレイヤー ] ボタンをクリックします。
  3. プレイヤーの名前を入力し、[ 保存] をクリックします。
  4. MEDIAモジュールを開きます。
  5. 公開するビデオをクリックします。
  6. プレーヤーのプロパティページで [ 公開と埋め込み] ボタンをクリックし、ドロップダウンから [ Web プレーヤー] を選択します。
  7. 実際のプレイヤーの場合は、作成したばかりのプレイヤーを選択します。
  8. [ コードをコピー] セクションで、[ 高度な実装] を選択します。
  9. 次の項目を設定します。
    • サイズ :固定
    • アスペクト比 :動画に合わせる
    • プレーヤーの幅 :希望の幅
  10. 最後に、コードをコピーします。
