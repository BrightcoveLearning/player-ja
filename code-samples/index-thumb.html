ページコンテンツ

    コードサンプル

    次のコードサンプルは、Brightcove Player のカスタマイズと機能強化を示しています。二重アスタリスク**でマークされたサンプルは、部分的または全体的にVideoCloudの機能に依存しています。 サンプルのアルファベット順のリストもありますここに。
     

    アウトストリーム (広告のみ) プレイヤー

    このサンプルは、サイズを動的に調整する方法を示すヘルパーJavaScript関数を提供します。<div>ページDOMの要素、そこにBrightcoveプレーヤーを挿入し、広告を読み込んで再生してから、プレーヤーを削除します

    プレーヤーを広告表示にスクロールする

    プレーヤーでは、水平ボリュームコントロールバーがデフォルトの垂直コントロールバーに置き換わっていることに注意してください。

    水平ボリュームコントロール（1.xプレーヤーのみ）

    プレーヤーでは、水平方向のボリュームコントロールバーがデフォルトの垂直方向のコントロールバーに置き換わります。

    垂直ボリューム

    デフォルトの水平レイアウトの代わりに垂直スライダーを表示するようにボリュームコントロールを設定します。

    プレイバックバーの広告インジケータ

    このサンプルは、ビデオに関連付けられた3つのアドキューポイントが青いマーカーとして再生バーに表示されているプレーヤーを示しています。

    SSAI プラグインによる広告

    この例では、VMAPファイルで定義されたIMA広告を使用して、ビデオストリームでサーバー側の広告を配信します。プレロール、ミッドロール、ポストロールの広告があります。VMAPファイルは広告設定で定義されています。

    広告カウントダウンタイマー

    オーバーレイプラグインを使用して、広告のカウントダウンタイマーを作成します。

     

    ModalDialogで年齢ゲート

    このサンプルでは、ModalDialogを使用して最初にプレーヤーをカバーしています。ユーザーがモーダルウィンドウをクリックして年齢を確認すると、コントロールが表示され、ビデオが再生されます。

    キューポイント表示CTA **

    プレーヤーは、キューポイントに基づいて表示される2つの異なるCTAを表示します。最初の CTA は 3 秒で、5 秒間表示されます。2番目のCTAは、13秒で3秒間表示されます。

    動的な行動を促すフレーズ**

    このプレーヤーでは、3つの異なる行動を促すフレーズの表示テクニックを見ることができます。

    iframe 親からビデオを再生/一時停止

    iframeの親ページの一部である再生ボタンまたは一時停止ボタンのいずれかをクリックして、iframeプレーヤーでビデオの再生/一時停止を開始することを確認します。

    エイジゲート

    このサンプルは、プレーヤーのコントロールと再生ボタンを非表示にするオーバーレイを表示します。ユーザーがオーバーレイをクリックして年齢を確認すると、コントロールが表示され、ビデオが再生されます。

    ミュート解除ボタンによる自動再生

    ビデオは、ビデオを実行するブラウザ/プラットフォームに応じて異なる動作をします。ビデオがオーディオで自動再生される場合は、自動再生されます。そうでない場合、ビデオの再生は開始されますが、ミュート解除ボタンが表示されます。

    背景ビデオランダムプレイリスト**

    プレーヤーには、再生リストからランダムに選択されたビデオが表示され、バックグラウンドビデオとして継続的に再生されます。各ビデオの開始時に、各ビデオのタイトルと説明が表示されます。

    背景ビデオ再生リスト**

    プレーヤーには、バックグラウンドビデオとして継続的に再生される4つのビデオがあります。また、各ビデオの開始時に、各ビデオのタイトルと説明が背景ビデオの上に表示されます。

    フルウィンドウの背景ビデオ**

    このトピックでは、プレイリストを使用して背景ビデオを提供する方法と、ブラウザーウィンドウのサイズが変更されたときにプレーヤーのサイズを自動的に変更して、ビデオがブラウザーウィンドウ全体に表示されるようにする方法を学習します。

    ビデオループの作成

    ループするビデオを使用したい場合があります。この例では、ビデオを3回ループしてから、ビデオの再生を停止します。

    プログレススクラバーを無効にする

    ビデオを再生します。プログレススクラバーを使用してビデオを進めることはできないことに注意してください。

    動画の開始時にページ上の他のプレイヤーを停止する

    ビデオの1つを再生し始めます。次に、別のプレーヤーでビデオを再生し、再生していた他のプレーヤーでビデオの再生が停止することを確認します。

    可変速度再生

    ビデオの再生を開始し、コントロールバーの再生速度ボタンを選択します（デフォルトは1x）。別のレートを選択して、ビデオの再生速度がどのように変化するかを確認します。

    プレーヤーのダイナミックロード

    このサンプルでは、[プレーヤーの追加]ボタンをクリックして、プレーヤーを動的にビルドおよびロードしてから、ビデオを再生します。

    RequireJSとブライトコーブプレーヤー

    このトピックでは、JavaScriptファイルおよびモジュールローダーであるRequireJSをBrightcovePlayerで使用する方法を学習します。

    ソースビデオを動的に変更**

    プレーヤーの下のボタンをクリックして、ソースビデオを動的に変更します。

    キオスクアプリ**

    プレーヤーは、ビデオオブジェクトの配列からビデオを無限ループで再生します。

    キャプションアニメーション

    ビデオの再生を開始します。マウスをプレーヤーの上に移動したりプレーヤーから離したりすると、キャプションがアニメーション化されてコントロールバーの上に留まります。

    関連リンクを表示**

    このサンプルでは、プレーヤーの下に表示されるビデオに関連するテキストが表示されます。Studioの[メディア]> [ビデオ情報]セクションを使用して、テキストと関連リンクを提供します。

    コントロールバーにビューを表示する

    ビデオが再生されると、コントロールバーにビデオビューが表示されます。情報が次のように表示されるまでに若干の遅延が発生する場合がありますAnalytics APIデータを非同期的に照会します。

    ホバー時にサムネイルを表示する

    この例では、ModalDialogを使用して、ビデオのさまざまなポイントから取得した4つのサムネイル画像を表示します。これにより、ユーザーはビデオの内容をスニークピークにすることができます。

    ランダムバンパーを表示する

    このサンプルでは、ランダムに選択されたバンパーが、再生リストの各ビデオの前に再生されます。この場合、バンパーは広告ですが、広告サーバーから描画されるのではなく、VideoCloudライブラリで定義されたビデオです。

    ハイパーリンクされた会社のロゴ

    この例では、プレーヤーのコントロールバーに会社のロゴを表示します。ロゴを選択すると、会社のWebサイトにリダイレクトされます。

    ライトボックスカルーセル

    カルーセル内のビデオをナビゲートします。サムネイルを選択して、ライトボックスコンテナ内のビデオを再生します。ライトボックスが表示された状態で、[閉じる] リンクを選択して、ライトボックスコンテナを非表示にします。

    スクロール付きライトボックス

    サムネイル画像の下にスクロールバーをドラッグして、ビデオ内を移動します。ビデオを選択して、ライトボックスコンテナ内のビデオを再生します。ライトボックスが表示された状態で、[閉じる] リンクを選択して、ライトボックスコンテナを非表示にします。

    シンプルなライトボックス

    この例は、BrightcovePlayerでライトボックスを開くインタラクティブなビデオポスター画像を表示します。ポスター画像を選択して、ライトボックスコンテナでビデオを再生します。

    ロゴオーバーレイプラグイン

    次の例では、プレーヤーの左下にロゴが表示されています。ビデオの5秒後に、ロゴはゆっくりとフェードアウトします。

    複数セッション再生

    ビデオを再生し、再生中のある時点で停止します。ページを更新すると、中断したところからビデオの再生が再開されます。

    ビデオ終了でナビゲート

    非常に短いビデオが完了すると、Brightcoveホームページへのリンクを含むオーバーレイが表示されます。ブライトコーブが提供するオーバーレイプラグインが使用されるため、JavaScript は必要ありません。CSSは、オーバーレイのスタイル設定に使用されます。

    プレーヤーを画面へスクロールする

    プレーヤーをスクロールして表示の内外に移動します。プレーヤーが完全にスクロールして表示すると、ビデオの再生が開始されます。プレーヤーをスクロールして非表示にすると、ビデオの再生が停止します。

    オーバーレイの切り替え

    この例では、インタラクティブオーバーレイの表示を切り替えることができます。[オーバーレイの切り替え]ボタンを選択して、オーバーレイを表示および非表示にします。

    カスタムイベントで巻き戻しをトリガーする

    ビデオの再生を開始します。次に、ビデオの巻き戻しボタンをクリックします。ビデオの再生が5秒戻るのがわかります。

    ビデオプラグインをダウンロード**

    コントロールバーの右側にダウンロードアイコンが表示され、プレーヤーに読み込まれたビデオの最高品質のMP4レンディションをダウンロードできます。

    ライブカスタムエラーメッセージ

    プレーヤーはライブストリームに接続できなくなります。ModalDialog オブジェクトが作成され、その中にカスタムイメージが配置され、ModalDialog オブジェクトが表示されます。ModalDialogを閉じると、デフォルトのエラーが表示されます。

    再生リストで最も視聴された動画**

    次の例では、プレイリストが右側に表示されているプレーヤーが表示されます。再生リストは、Brightcove Playerカタログを使用して、最も人気のある動画を再生回数に基づいて取得することで作成されます。

    再生リストの最新動画**

    次のプレーヤーには、ナレッジチームのアカウントに追加された最新の10本の動画を含む再生リストが表示されます。

    再生APIページング**

    この例では、Brightcoveプレーヤーを使用して、プレイリストの最初の3つのビデオとともにプレーヤーにビデオを表示します。[次の動画]ボタンを選択して、再生リストの次の3つの動画を表示します。

    人気の動画オーバーレイ**

    動画を一時停止するか、動画が終了すると、オーバーレイが表示されます。過去 24 時間で最も多く視聴された 6 本の動画を API 応答とともに取得するためのリクエストが表示されます。Analytics API

    人気の動画再生リスト**

    ビデオデータが取得されるとすぐに、プレイリストが自動的に表示されます。過去 24 時間で最も多く視聴された 6 本の動画を API 応答とともに取得するためのリクエストが表示されます。Analytics API

    再生リストの前のビデオ

    再生リスト対応のプレーヤーを作成して、再生リストにない動画から始めます。

     

    再生リストの最後の動画

    再生リストに3つの動画が含まれていることがわかります。これら 3 つの動画が再生された後、プレイリストにはない別の動画が再生されます。再生リストにないビデオが再生された後、ビデオの再生は停止します。
