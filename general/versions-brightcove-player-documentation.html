Brightcove Playerドキュメントのバージョン

{{ page.description }}

Brightcove Playerのドキュメントには、次の2つのバージョンの情報が含まれています。

  • ビデオクラウドアカウントで Brightcove プレーヤーを使用する
  • Brightcove Playerをスタンドアロン（実行）モードで使用する（他のソースからのコンテンツを使用）
ほとんどのコンテンツは両方のバージョンで同じであるため、他のバージョンに切り替えたときに目に見える変化が見られないことがあります。

どちらの場合も同じプレーヤーですが、Video Cloudアカウントをお持ちの場合、プレーヤーはビデオとアセットの場所をVideo Cloudから直接プルできるため、その情報を手動で提供する必要はありません。

これが違いを生む場合は、シナリオごとにコンテンツの代替バージョンを作成しました。状況に合ったシナリオに応じてバージョンを選択する必要があります。

ページの右上に表示されるボタンを使用して、Video Cloud バージョンとスタンドアロンの Player バージョンのドキュメントを切り替えることができます。
