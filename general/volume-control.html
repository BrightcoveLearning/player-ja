ボリュームコントロール

{{ page.description }}
注:以下の例では、volume()そしてmuted()メソッドは、プレーヤーの準備ができたときに実行されます。たとえば、ボリュームはプレーヤー設定であり、再生中のビデオとは独立しているため、metatdataloadedイベントを待つ必要はありません。

volume（）メソッド

volume()このメソッドは、引数が渡されたかどうかに応じて、ゲッターとセッターの両方として機能します。メソッドをセッターとして使用する場合、引数は 0 (ミュート) から 1.0 (ボリューム全体) の間の 10 進数です。引数は 10 進数で表されるパーセンテージです。

以下は、volume()メソッドを使用して、ボリュームを 60% に設定するコード例を示しています。

  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.volume(.6);
  });

ミュート () メソッド

muted()このメソッドは、引数が渡されたかどうかに応じて、ゲッターとセッターの両方として機能します。メソッドがセッターとして使用される場合、引数はブール値で、truefalseミュートとミュート解除を行います。

以下は、muted()プレーヤーのオーディオをミュートするメソッドを使用してプログラム的にコード例です。

  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.muted(true);
  });
注:間に微妙な違いがありますvolume(0)そしてmuted(true)。違いは、ミュートを切り替えると、音量は以前のパーセンテージのままになることです。muted()使われた。もちろん、音量が0に設定されている場合、オーディオを再開するにはパーセンテージを再度設定する必要があります。

volumechangeイベント

音量の変化に反応したい場合は、volumechangeイベントを使用できます。イベントリスナーは次のように設定できます。

    myPlayer.on('volumechange', function( evt ){
      console.log('event: ', evt);
    })
