ブライトコーブのプラグイン

以下は、ブライトコーブが提供するプレーヤープラグインの概要です。プラグインは、Brightcove Playerを強化するために使用されます。

プラグイン デフォルトで読み込まれた 説明
360º ビデオ   このプラグインは、適切に録画されたビデオ要素を、Brightcove Playerで再生できる HTML5 パノラマの 360 度ビデオに変換します。
広告のみ   このプラグインを使用すると、プレーヤーでIMA3広告のみを再生できます。通常のビデオコンテンツは提供されていません。
FreeWheelで広告   このプラグインは、Brightcove Playerで Flash または HTML5 の FreeWheel 広告テクノロジーを有効にします。
IMA3で広告   HTML5 バージョン 3 のBrightcove Playerと Google のインタラクティブメディア広告 (IMA) を統合します。これにより、プレイヤーの VAST 広告をリクエストして追跡できます。
SSAIとの広告   サーバーサイド広告挿入（SSAI）では、ブラウザーの広告ブロッカーによってブロックされないように広告を動画に埋め込むことができます。デフォルトでは、プラグインはすべての広告が監視され、再生中に広告カウントダウンタイマーが表示されます。このプラグインを簡単にカスタマイズして広告をスキップできます。
AirPlay   Brightcove Playerから Apple TV へのコンテンツのストリーミングを許可します。
Chromecast   Brightcove Playerがデスクトップまたは Android Chrome ブラウザから Chromecast デバイスにビデオをキャストできるようにします。
カスタムエンドスクリーン   ビデオ再生の終了時に、パブリッシャーが提供した HTML を表示します。
サムネイルプレビューを表示   ビデオ内の定義したポイントで、タイムライン上にプレビューイメージを表示します。ユーザーがタイムライン上を移動すると、指定した時間範囲でサムネイル画像が表示されます。
DRM（正式名称は DASH）   Brightcove Playerで DRM を実装するために使用します。
エラーメッセージ Y エラーが発生したときに、プレイヤーにわかりやすいメッセージを表示できるようにします。表示は、半透明のオーバーレイで、既定のスタイルシートでスタイル設定されます。
グーグルタグマネージャ   Google タグマネージャー機能と Brightcove Player との統合を可能にします。
HLS Y HLS をサポートしていないが Flash Player を搭載しているプラットフォームで HLS ビデオを再生します。では、m3u8 マニフェストのビデオコンテンツをプレーヤーで再生できます。
オーバーレイ表示   シンプルなメッセージを半透明のオーバーレイとしてプレーヤーの上に表示します。メッセージの表示は、再生や一時停止などのプレーヤーイベント、またはビデオ再生中の特定の時間に基づいて行うことができます。
ピクチャー・イン・ピクチャー   ユーザーがページをスクロールダウンでき、プレイヤーは Web ページの隅に固定されます。
プレイリスト終了画面   このプラグインは、ビデオの最後にプレイリストを表示できるようにします。これをプレイヤーに追加するには、Video Cloud Studio を使用するか、Brightcove Player でコードを使用します。
プレイリスト UI   デフォルトのプレイリスト機能のルックアンドフィールを強化します。
品質選択   プレイヤーのコントロールバーにメニューボタンがあり、HLS または Dash ソースの再生品質を手動で選択できます。
ソーシャルメディア   ユーザーがFacebook、ツイッター、Tumblr、Pinterest、または LinkedIn でビデオを共有できるようにします。
テアリウム   Tealium iQ タグマネージャーとの統合を可能にします。
サムネの求め   Brightcove Playerでサムネールシークを実装します。

オープンソースプラグイン

Brightcove Playerのベースとなるオープンソースの Video.js プレーヤーには、多くのプラグインが構築されています。これらのプラグインは、Video.js プレーヤーのバージョンがBrightcove Playerのベースと同じバージョンである限り、Brightcove Playerと互換性があります。Brightcove Player は、オープンソースの Video.js プレーヤーの前後または後方のバージョン単位でもかまいません。Video.js プラグインページへのリンクは https://github.com/videojs/video.js/wiki/Plugins です。

ブライトコーブ Player 固有のオープンソースプラグインも利用できます。例としては、Conviva と Nielsen アナリティクスのプラグインがあります。ブライトコーブは、これらのプラグインのサポートやドキュメントを提供していないことに注意してください。詳細とアクセスについては、Conviva/Nielsen のアカウントマネージャーにお問い合わせください。

再生された広告の視認性を簡単に測定してレポートするパブリッシャーを提供する会社であるMoatは、Brightcove Playerと連携するプラグインです。ブライトコーブ Player 用の Moat のプラグインにアクセスするには、お客様が Moat アカウントマネージャーに連絡する必要があります。
