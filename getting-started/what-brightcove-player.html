Brightcove Playerとは何ですか？

{{ page.description }}

主な特徴

  • HTML5 -Brightcove Playerの UI は完全に HTML5 で書かれており、モバイルデバイスとデスクトップデバイス間で一貫した外観と動作を実現します。
  • レスポンシブ -プレイヤースキンはデフォルトでレスポンシブで、ビデオのアスペクト比を維持しながら、変化するウェブレイアウトに合わせて自動的にサイズが変更されます。
  • 高速読み込み -瞬時にロードされ、最初のフレームまでの時間が非常に高速で、視聴者はすぐにコンテンツと関わります。
  • 簡単なカスタマイズ -標準のウェブテクノロジー（CSS と JavaScript）を使用して、プレイヤーのほぼすべての詳細をカスタマイズできます。
  • クロスプラットフォーム HLS -ネイティブの HLS 機能を持たないプラットフォームでも高画質の適応ビットレートビデオは、最小のダウンロードで最高品質を提供します。
  • プレーヤー管理 -Video Cloud またはプレーヤー管理 API を使用して、プレーヤーの設定を簡単に追跡し、スタイルやプラグインに変更を加えることができます。各ページのコードを変更することなく、サイト全体でプレイヤーを簡単に更新できます。
  • 広告とアナリティクスの統合 -Brightcove Playerは、Google IMA3、Freewheelなどを含む人気のある広告および分析システムと統合するためのプラグインが多数用意されています。
