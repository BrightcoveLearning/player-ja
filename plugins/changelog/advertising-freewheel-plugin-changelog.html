注:実際のFreeWheelプラグインのドキュメントに記載されているように、正しいSDKURLを使用するセクションでは、FreeWheelは動的SDK URLを廃止し、バージョン管理されたURLを採用しました。つまり、Brightcove Player プラグインはデフォルトで最新バージョンではなく、特定のバージョンにロックされています。ブライトコーブは FreeWheel SDK リリースに追いつこうとしますが、遅れる可能性があります。

Brightcove Player インテグレーションでは、独自の SDK バージョンを管理することができます。これには、Html5.sdkurlオプションを FreeWheel プラグインに渡すことが含まれます。

player.FreeWheelPlugin({
  Html5: {
    sdkurl: 'preferred-sdk-url'
  }
});
