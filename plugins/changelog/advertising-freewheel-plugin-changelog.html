    • 次の場合に広告コンテナのサイズを変更しましたplayerresizeイベントが発生します

    • VPAID広告のサイズ設定の問題を修正しました
    注：実際のFreeWheelプラグインのドキュメントに記載されているように、正しいSDKURLを使用するセクションでは、FreeWheelは動的SDK URLを廃止し、バージョン管理されたURLを採用しました。これは、Brightcove Playerプラグインが、最新バージョンではなく、デフォルトで特定のバージョンにロックされるようになったことを意味します。Brightcoveは、FreeWheel SDKのリリースについていくように努めますが、遅れる可能性があります。

    Brightcove Player インテグレーションでは、独自の SDK バージョンを管理することができます。これには、Html5.sdkurlオプションを FreeWheel プラグインに渡すことが含まれます。

    player.FreeWheelPlugin({
  Html5: {
    sdkurl: 'preferred-sdk-url'
  }
});
