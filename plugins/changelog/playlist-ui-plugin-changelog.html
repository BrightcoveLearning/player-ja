プレイリスト UI プラグインの変更履歴

以下は、プラグインの変更を追跡します。

更新履歴

2020年5月7日

v3.8.0

  • 再生リストのサムネイルに動画の説明を表示するオプションを有効にする

2020年2月10日

v3.7.0

  • アラビア語の翻訳を追加しました
  • 次のオーバーレイ使用時に初期化中にスローされるエラーを修正しました

2019年10月9日

v3.6.3

  • フォーカスがあるときにプレイリストタブを表示する

2019年10月8日

v3.6.2

  • Firefox と IE11 の水平再生リストに関するフォローオン UI の問題を解決しました

2019年9月9日

v3.6.1

  • IE11とFirefoxで水平再生リストが正しく表示されない問題を修正

2019年8月27日

v3.6.0

  • 中国語、ドイツ語、スペイン語、フランス語、日本語、韓国語の翻訳を追加しました

2019年7月16日

v3.5.0

  • コンテナ要素を介したページ内プレーヤーの iframe のようなプレイリスト UI のサポートを追加

2019年3月20日

v3.4.4

  • videojs-Playlist を 4.3.1 に更新。これにより、videojs-Playlist 4.3.0 で導入されたリグレッションが解決されます

2019年1月14日

v3.4.3

  • videojs-playlist を 4.3.0 に更新。これにより、同じソース URL で複数の項目を含むプレイリストではまれな問題が解決されました

2018年10月29日

v3.4.2

  •  playlistPickerプレイリストピッカーの削除を許可するオプション（デフォルトはtruefalse )
  • バグ修正:iframeプレーヤーの水平プレイリストを、可変の高さではなく、静的20％の高さにする
  • バグ修正:水平再生リストに垂直スクロールバーが表示される問題を修正
  • バグ修正:プレイヤーが処分されたときにプレイリストUIを削除する

2018年10月3日

v3.4.1

  • 水平再生リストに垂直スクロールバーが表示される問題を修正
  • プレーヤーが破棄されたときにプレイリスト UI を削除する

2018年9月25日

v3.4.0

  • PlayListPicker オプションを追加しました。デフォルトは true で、プレイリストピッカーを削除できます。

2018年9月17日

v3.3.2

  • プレイリストプレーヤーが間違ったタイミングにロードされると、UIがレンダリングされない問題を修正
  • インストールの問題を防ぐために、ポストインストールスクリプトを削除しました
  • playプレイリストで約束の例外を黙らせました
  • プラグインジェネレータ v7.2.0 を使用して更新しました。

2018年6月13日

v3.3.1

  • 次のアイテムボタンにポインタカーソルがない問題を修正

2018年5月23日

v3.3.0

  • 属性を介してプレイリスト要素をプレーヤーと明示的に関連付けるサポート

2018年5月9日

v3.2.1

  • プレイリストを持つ複数のプレーヤーがあるページで、プレイリストコンテナの前にスクリプトを配置する埋め込みコードにより、それらのプレイリストが順番から外れる問題を修正しました。

2018年4月9日

v3.2.0

  • プレイリストが変更されるたびにランダム化するシャッフルオプションを追加しました

2018年3月29日

v3.1.1

  • オーバーフローしたときに長いビデオタイトルを省略記号で切り捨てる

2018年1月25日

v3.1.0

  • videojs-Playlist 4.2.0 duringplaylistchangeにアップデートしrest、shuffle()メソッド

2018年1月17日

v3.0.1

  • プレイリストが空だったのに、iframe のプレイリストのサイドバーにまだ表示されるケースに対処しました
  • イベント終了後にユーザーが再生を再開しても自動アドバンスが発生する問題を修正
  • モバイルで次のオーバーレイとエンドスクリーンでアイコンをクリック/タップする問題を修正

2017年12月22日

v3.0.0

  • 特徴
    • オプションの Up Next 閉鎖可能なオーバーレイカードを追加
    • オプションの Up Next エンドスクリーンモーダルを追加
    • プレイヤーの関連するプレイリスト要素をよりインテリジェントに見つけることで、複数のページ内プレイヤーのサポートが向上
    • ビデオプレイリストを直接の依存関係としてバンドルする
    • iframe プレーヤーで水平プレイリストをサポートする
  • コードリファクタリング
    • サイドバーをコンポーネントに移動し、リファクタリングプラグインをクラスベースにする
    • 新しいリファクタリングアップ次機能
    • ツールをプラグインジェネレータ v5.1.1 に更新する
  • 変更を破る
    • IE8 のサポートを削除
    • Video.js/brightCove Player v5をサポートしなくなりました
