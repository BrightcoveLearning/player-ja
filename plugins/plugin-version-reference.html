HTML で使用する

HTML ページでプラグインの URL を直接使用する場合は、JavaScript <script>用のタグと CSS <link>用のタグを使用します。例を次に示します。

<script src="https://players.brightcove.net/videojs-vr/1/videojs-vr.min.js"></script>
<link href="https://players.brightcove.net/videojs-vr/1/videojs-vr.css" rel="stylesheet">
