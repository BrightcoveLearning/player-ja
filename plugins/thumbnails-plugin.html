    ブライトコーブプレーヤーでのサムネイルのスクラブ

    このトピックでは、Brightcove Player でサムネイルスクラブを実装する方法について説明します。
    この機能を使用するには、再生 API v2 を使用する必要があります。この機能を利用できるかどうかについては、アカウントマネージャーにお問い合わせください。

    概要

    ビデオサムネイル画像を使用すると、ユーザーはプログレスバーをすばやくスキャンして、関心のあるセクションを探すことができます。プログレスバーをクリックすると、ビデオ内のその場所に移動します。

    アカウントで最初にサムネイルスクラブを有効にする必要があります。これが完了すると、ユーザーがプログレスバーにカーソルを合わせるか、再生ヘッドをプログレスバー上の場所にドラッグすると、BrightcovePlayerにサムネイル画像が表示されます。

    要件

    サムネイルプラグインには、次の要件が適用されます。

    • BrightcovePlayerバージョン6.41.0以降を使用する必要があります
    • 動的配信には動画が取り込まれなければならない
    • サーバー側の広告挿入（SSAI）の場合、 videojs-ssaiプラグインバージョン1.10.0+

    プレーヤーの例

    以下の例をクリックし、進行状況バーに沿ってホバーまたはスクラブしてプレビューサムネイルを表示します。

    ペンを見るサムネイルプラグイン Brightcove Learning Services（ @ bcls1969） オンCodePen。


    ソースコード

    ソースコードを表示するには、 HTML上のCodePenのボタンをクリックするか、 GitHubの完全なソリューション。

    入門編

    この機能は、Brightcove 動画クラウドアカウントをお持ちの方であれば誰でも利用できます。

    動画が動的配信用に取り込まれていることを確認します。

    Studioを使用したサムネイルの実装

    サムネイルスクラブ用にプレーヤーを構成する最も簡単な方法は、Video CloudStudioを使用することです。

    1. を開きますプレイヤーズモジュール。既存のプレーヤーを使用するか、新しいプレーヤーを作成します。

      BrightcovePlayerバージョン6.41.0以降を使用することを忘れないでください。
    2. プレーヤーのリンクを選択して、プレーヤーのプロパティを開きます。
    3. 左側のナビゲーションで、プラグイン。

    4. を展開しますプラグインを追加するドロップダウンメニューをクリックし、 Brightcoveプラグイン。

    5. を展開しますBrightcoveプラグインドロップダウンメニューをクリックし、サムネイル。

    6. [ 保存 ] ボタンを選択します。これで、プラグインがプレーヤーに追加されたことがわかります。

    7. プレーヤーを公開するには、公開と埋め込み>変更の公開

    8. これで、サムネイルプラグインがプレーヤー用に構成されました。

      動的配信用に取り込まれたビデオを使用し、上記の手順で構成したプレーヤーで公開します。プレーヤーのプログレスバーにカーソルを合わせるか、スクラブすると、サムネイル画像が表示されます。

    WebVTTサムネイルの実装

    独自のサムネイル画像を使用することを選択した場合は、カスタムWebVTTファイルを作成し、それを使用してプレーヤーに渡すことができます。player.addRemoteTextTrack()方法。手順は次のとおりです。

    1. 作成する.vtt標準のWebVTT形式に従うファイル。キャプションテキストを各サムネイル画像のURLに置き換えます。次のようになります。

    2. Video Cloud Studioで、動的配信用に取り込まれたビデオを選択します。バージョン6.41.0以降のプレーヤーの高度な埋め込みコードをコピーします。
    3. WebページのHTMLファイルに、高度な埋め込みコードを貼り付けます。
    4. の中にhead HTMLコードのセクションにサムネイルCSSファイルを追加します。 
      //players.brightcove.net/videojs-thumbnails/1/videojs-thumbnails.css
    5. の中にbody HTMLコードのセクションに、サムネイルJavaScriptファイルを追加します。 
      //players.brightcove.net/videojs-thumbnails/1/videojs-thumbnails.js
    6. にvideoタグ、追加idプロパティ。
    7. 閉会前bodyタグ、追加scriptブロック。

    8. 作成したスクリプトブロックに、以下を追加します。

      プロパティ 値
      type ビデオのmimeタイプ
      src 動画のソースURL

      あなたのコードは次のようなものでなければなりません： 

      var player = bc('myPlayerID');

player.thumbnails();

player.ready(() => {
  player.src({
    type: 'video/mp4',
    src: '//solutions.brightcove.com/bcls/videos/Great Blue Heron.mp4'
  });
});

    9. 同じスクリプトブロックに、以下を追加します。

      プラグインが初期化され、プレーヤーの準備ができたら、サムネイル画像ソースをWebVTTファイルでプラグインに渡します。addRemoteTextTrack()方法。サムネイルファイルを使用してメソッドを呼び出すときは、次の設定も行う必要があります。

      プロパティ 値
      src サムネイル画像を含むWebVTTファイルの場所
      kind に設定metadata
      label に設定thumbnails
      mode に設定hidden
      addRemoteTextTrack() manualCleanup に設定falseソースが変更されたときにトラックが自動的に削除されるようにします

      あなたのコードは次のようなものでなければなりません： 

      var player = bc('myPlayerID');

player.thumbnails();

  player.ready(() => {
    player.src({
      type: 'video/mp4',
      src: '//solutions.brightcove.com/bcls/videos/Great Blue Heron.mp4'
    });

  player.addRemoteTextTrack({
    src: 'thumbnails.vtt',
    kind: 'metadata',
    label: 'thumbnails',
    mode: 'hidden'
  }, false);
});
