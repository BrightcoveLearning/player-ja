シンジケーション戦略

{{ page.description }}

オープンでバイラルなシンジケーション

可能な限り幅広い配信を実現できる動画に満足している場合は、オープンシンジケーションアプローチを採用できます。プレイヤーを公開したり、訪問者がプレーヤーの公開コードを取得し、そのプレーヤーを Web サイトに埋め込むことができる Web ページを作成できます。

オープンシンジケーションのサブセットはウイルスシンジケーションです。Brightcove Player には、視聴者がプレーヤーをクリックして Facebook、Twitter、その他のソーシャル共有サイトで動画を共有したり、プレーヤーの埋め込みコードやプレーヤーへの URL リンクを取得したりする機能が用意されています。これにより、プレーヤーや動画コンテンツを任意の Web サイトやメールで表示できます。プレーヤーのソーシャルプロパティの設定についてもっと読んでください。

バイラルシンジケーションは、視聴者を拡大するには非常に効果的なツールですが、これを利用することにより、いくつかのコントロール機能を失うことにもなります。オープンシンジケーションには、Web サイト、またはアフィリエイト' Web サイトで作成したコンテンツから動画を取り出すという欠点があります。あなたのブランディングがビデオと一緒に旅行することを保証するためにあなたが取ることができるステップがあります-参照してください透かしフィールド取り込みプロファイルで。

信頼できるパーティーシンジケーション

信頼できるパーティーシンジケーション（マネージドシンジケーションとも呼ばれる）では、アフィリエイトとの契約関係を構築します。この関係は、多くの場合、利用可能なコンテンツに関する合意を伴い、コンテンツの支払いや広告収益を分割する契約が含まれる場合があります。広告が動画ビジネス戦略の一部である場合は、同時配信動画に関連して広告を掲載する権利の予約について、シンジケーションパートナーと明確な合意があることを確認してください。信頼されたパーティーシンジケーションを使用すると、コンテンツを表示するユーザーとそのコンテンツの公開場所を制御できます。

信頼できるパーティーシンジケーションのプレイヤー

信頼できるパーティーシンジケーションアプローチを採用している場合は、シンジケーションパートナーごとに専用の Brightcove Player を作成することをお勧めします。この戦略では、プレイヤーごとにパフォーマンスを記録し、プレーヤーごとにコンテンツや広告キャンペーンをターゲットに設定できます。また、プレーヤーに公開されるコンテンツも制御でき、Studio 内からいつでもコンテンツを更新または削除できます。この戦略を実装する主な手順は次のとおりです。

  1. ブライトコーブのアカウントで、シンジケーションパートナーごとに個別のプレーヤーを作成します。あなたはできるカスタマイズパートナーからの入力で、あなたが望むすべてです。
    どのアフィリエイトがどのプレイヤーと一緒になるかを簡単に識別できる命名規則を採用し、シンジケートされたプレイヤーを自分のサイトのプレイヤーから分離できるようにします。
  2. プレーヤーをアフィリエイトのウェブサイトに限定するためのドメイン制限を使用します。これにより、アフィリエイトのサイトから埋め込みコードを暗くし、別のサイトに公開することはできません。他のプレイヤーの設定を調べます。プレーヤーのホスティングオプションを構成して、リンクによってトラフィックがサイトに戻ったり、プレーヤーの広告をオフにしたりできます。Video Cloud が提供するプレーヤー設定を考えてみて、シンジケーションパートナーとの契約を反映するようにプレーヤーを設定します。
  3. プレーヤーのコンテンツを選択します。プレイリストプレーヤーの場合は、各アフィリエイトのプレーヤーに表示される動画が随時制御されます。または、シングルビデオプレーヤーの場合は、クイックビデオ公開を使用するか、プレーヤー公開コードを変更してプログラムできます。
  4. プレーヤー公開コードを配布します。プレーヤーコードをシンジケーションパートナーに送信し、そのパートナーは自分のウェブページに特定のプレーヤーを埋め込むことができます。標準（iframe）または詳細（ビデオタグ）の埋め込みコードを使用できます。ただし、標準の埋め込みコードを配布すると、アフィリエイトがそのサイズとは別にプレーヤーを変更できなくなるため、プレーヤーのより厳密な制御を維持することができます。
  5. パフォーマンスデータを監視します。あなたはあなたのビデオが各アフィリエイトのサイトでどのように実行されているかを知りたいです。これは、広告を掲載している場合や、同時配信コンテンツの収益を共有する場合に特に重要です。アフィリエイトごとにプレーヤーを作成しているので、Video Cloud Analytics でアフィリエイトごとのパフォーマンスを簡単に確認できます。
