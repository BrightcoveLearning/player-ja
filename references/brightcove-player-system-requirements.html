Brightcove Playerシステム要件

このトピックでは、Brightcove Playerでサポートされるプラットフォームとブラウザーについての情報を提供します。
ブラウザ バージョン 備考
Internet Explorer (Win) 11 2021 年 6 月 15 日以降、ブライトコーブはどのバージョンの Windows でも Internet Explorer 11 (IE11) を積極的にサポートしなくなります。この日以降、関連するバグレポートと拡張リクエストは受け付けられません。Brightcove Player 6.x のどのバージョンでも IE11 サポートを意図的に削除することはしませんが、このサポートに対する積極的なテストも行いません。IE11のサポートが必要なお客様は、機能を継続するために、6月15日より前にリリースされた最後のバージョンにプレーヤーをロックすることをお勧めします。そのバージョンは6.56.0です。ロックされたプレイヤーは、新しい拡張機能やバグ修正を受け取らないことに注意してください。Edge は Microsoft の推奨ブラウザで、ブライトコーブチームが完全にサポートしています。
Microsoft Edge (Win & Mac) 現在のメジャーバージョンと以前の2つのメジャーバージョン  
Chrome デスクトップ (Win、Mac) 現在のメジャーバージョンと以前の2つのメジャーバージョン  
Chromeモバイル (Android) 現在のメジャーバージョンと以前の2つのメジャーバージョン GoogleはもはやChromeブラウザでAndroid 4.x以前をサポートしていません。
Chrome モバイル (iOS) 現在のメジャーバージョンと以前の2つのメジャーバージョン プラグインの互換性については、Brightcoveの IMA またはその他のサードパーティのドキュメントを参照してください。
Safari デスクトップ（Mac） 現在のメジャーバージョンと以前の2つのメジャーバージョン  
Safari モバイル (iOS) 現在のメジャーバージョンと以前の2つのメジャーバージョン  
Firefoxデスクトップ（Win、Mac） 現在のメジャーバージョンと以前の2つのメジャーバージョン  
Firefox ESR 現在  

 

注意:

    \
  • Windows 8.1+のIE 11では、HTMLベースのHLSとDASH（MSEを活用）を使用している。
  • Windows < 8.1 の IE 11 では HLS にはFlash (Flashが無効になっている場合は MP4 にフォールバックします)、DASHには Silverlight を使用します。Flashと Silverlight は、マイクロソフトではサポートされなくなったため、利用できない場合があります。
  • 互換モードでは、IE 10 および 11 は、ドメイン制限コンテンツを表示するときにドメイン制限エラーを表示します。
  • IE の互換モードはサポートされていません。
  • IEのエンタープライズモードはサポートされていません。
  • Windows 10N/10KN エディションには、Windows Media Player および関連テクノロジがプリインストールされていません。ビデオを再生するには、 マイクロソフトのメディア機能パックをインストールすることができます。
  • 実機のみがサポートされます。通常、プレーヤーは仮想マシンとホスト仮想デスクトップ (HVD) で動作すると予想されますが、これらのテストは行わず、サポートしていません。
  • Flash Playerが必要な場合、最新バージョンが必要です。
  • Androidの以前のバージョンで見つかった「ストック」ブラウザ（多くの場合、「インターネット」という名前のアイコン）には、ビデオ再生に関する多くの問題があり、推奨されません。
  • Prototype.js v1.6 が存在する場合、Brightcove Playerが失敗することが知られており、v1.7 が存在する場合は動作する可能性があります。Brightcove Player は、Prototype.js のどのバージョンでも共有環境での使用をサポートしていません。
