    ブライトコーブプレーヤーリリースノート

    このページは Brightcove Player のリリース情報を追跡しています

    注:デフォルト設定として、更新内容はすべてのプレーヤーに自動的に適用されます。プレーヤーの自動更新を防ぐには、手動更新 (プレーヤーのロックとも呼ばれます) に設定する必要があります。これは、 Studio Players モジュール (外部サイト) またはプレーヤー管理 API を使用して行うことができます。

    自動更新をオプトアウトすることは、更新プログラムによって作成される問題を解決するときに適用される一時的な措置にする必要があります。プレーヤーを現在のバージョンに更新しないと、新機能やバグ修正が欠落しており、古いバージョンのプレーヤーは一部の時点でサポートされなくなります

    また、Brightcove Playerのリリース、およびプラグインのリリースは、 Brightcoveプレーヤーリリースのお知らせ Googleグループ。
