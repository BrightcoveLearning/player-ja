プレイリストのスタイリング

このトピックでは、プレイリストのスタイルを設定する方法を学習します。

概要

プレイリストの外観を変更する場合は、CSS を使用して変更できます。このドキュメントでは、変更できるスタイルの数が表示されます。もちろん、可能な変更は多数あり、タスクを実行するには正しいセレクターを見つける必要があります。

次のドキュメントは、プレイリストの使用の全体像を完成させるために不可欠な情報です。
  • プレイリストの実装 :このドキュメントでは、プレイリストの使用に関して行う必要がある決定と、HTML ページで実際にプレイリストを取得するために必要なプレーヤーコードを取得する方法について説明します。
  • プレイリスト UI プラグイン :このドキュメントでは、プレイリスト機能のルックアンドフィールを強化するために利用できるオプションについて詳しく説明します。

垂直プレイリストのスタイリング

次の図は、垂直再生リストに対して変更されたスタイルをまとめたものです。図の後に、スタイルをリストした表があります。

プレイリストのテキストの色をコントロールする .vjs-mouse.vjs-playlist {
  色:#E9D2B3;
}
の色をコントロールする
プレイリストアイテム間のスペース		 .vjs-mouse.vjs-playlist {
  background-color: #161D6E;
}
デュレーションテキストの色をコントロールする .vjs-playlist .vjs-playlist-duration {
  色:#9BB4DA;
}
デュレーションの背景の色をコントロールする .vjs-playlist .vjs-playlist-duration {
  background-color: #1160AA;
}
プレイリストアイテム間のスペースをコントロールする .vjs-playlist-vertical .vjs-playlist-item {
  マージントップ:20px;
}
最初のプレイリスト項目の上に余白が付かないようにするには、マージン設定スタイルを次のように指定する必要があります。 
.vjs-playlist-vertical .vjs-playlist-item:first-child {
  margin-top: 0px;
}

水平プレイリストのスタイリング

次の図は、水平再生リストに対して変更されたスタイルの概要を示しています。図の後に、スタイルをリストした表があります。

プレイリストのテキストの色をコントロールする .vjs-mouse.vjs-playlist {
  色:#E9D2B3;
}
の色をコントロールする
プレイリストアイテム間のスペース		 .vjs-mouse.vjs-playlist {
  background-color: #161D6E;
}
デュレーションテキストの色をコントロールする .vjs-playlist .vjs-playlist-duration {
  色:#9BB4DA;
}
デュレーションの背景の色をコントロールする .vjs-playlist .vjs-playlist-duration {
  background-color: #1160AA;
}
プレイリストアイテム間のスペースをコントロールする .vjs-playlist-水平.vjs-playlist-item {
  マージン左:5px;
}
最初のプレイリスト項目の左側に余白が表示されないようにするには、マージン設定スタイルに次のものを使用する必要があります。 
.vjs-playlist-horizontal .vjs-playlist-item:first-child {
  margin-left: 0px;
}

スクロールバーを非表示にする

水平再生リストのスクロールバーは必要ありませんが、表示されます。スクロールバーを非表示にしたい場合は、次のスタイルがそれを行います：

div.vjs-playlist.vjs-playlist-horizontal {
  overflow-y: hidden;
}
