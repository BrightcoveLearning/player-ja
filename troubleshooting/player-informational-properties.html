追加情報

プレーヤーのプロパティ

Brightcove Player プロパティには、特定のプレーヤーに関する情報が含まれます。プレイヤースキンとバージョンの設定の詳細については、スキンとプレーヤーのバージョンのドキュメントを参照してください。

版

bc.VERSIONプロパティは、プレーヤーの設定で定義されているプレーヤーのバージョンを指定します。

ここに例があります：

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="https://players.brightcove.net/1752604059001/default_default/index.min.js"></script>


    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
戻り値:

この場合に返される値は 5.1.7

この値は、プレーヤーの設定から得られます。この値は、ブラウザでプレーヤーをプレビューし、index.htmlに変更することで確認できますconfig.json。

肌

bc.SKINプロパティは、プレーヤーに適用されるスキンを指定します。

ここに例があります：

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="//players.brightcove.net/1752604059001/default_default/index.min.js"></script>

    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	  videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
戻り値:

この場合に返される値は null です。つまり、Luna スキンがこのプレイヤーに適用されます。bc.SKINプロパティからの戻り値の概要を次に示します。

BC.Skin 値 スキンを適用
偽 スキンを使用しないので、独自のスタイルでオーバーライドできます
グラファイト 黒鉛皮膚
未定義、ヌル、真などを含む他のすべての値（またはその欠如） ルナスキン (既定のスキン)

この値は、プレーヤーの設定から得られます。この値は、ブラウザでプレーヤーをプレビューし、index.htmlに変更することで確認できますconfig.json。

注:ザ・bc.SKINプロパティは、"skin"プレーヤー構成のプロパティには明示的な値があります。それ以外の場合、未定義になります。
