構成の表示

プレイヤーの設定を表示するには、次の手順に従います。

  1. Studioで、プレーヤーモジュールに移動します。
  2. 設定を見たいビデオをクリックします。
  3. [ パブリッシュと埋め込み ] ボタンをクリックします。
  4. すべての変更が公開されていることを確認してください。
  5. クリックしてくださいプレーヤーのURL  （以下に示すように）プレーヤーがブラウザーで開いているのがわかります。
  6. ブラウザでURLを確認すると、次のように表示されます。 
    https://players.brightcove.net/1752604059001/SAJo2wjVo_default/index.html
  7. 変更index.htmlへのURLでconfig.json。
  8. 変更されたURLには、プレーヤーの構成が表示されます。
    注:結果のJSONは、スクリーンショットに示されているほどうまく表示されない場合があります。JSON のフォーマットに役立つブラウザの JSON フォーマッタ拡張機能/アドオンを見つけることができます。

構成の変更

構成の変更はこの文書の範囲を超えていますが、そのテーマに関するコメントが適切です。

  • Studioは、プレーヤーの構成の多くの側面を変更するためのUIを提供します。プレイヤーモジュール。
  • プレイヤーの設定を完全に制御するために、プレーヤー管理 API を使用できます。これは、開発者向けであり、 curlステートメントを広範囲に使用していることに注意してください。ステップバイステップ:プレイヤー管理ドキュメントでは、プロセスの概要を説明します。
